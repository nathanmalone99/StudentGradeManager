# Student Grade Manager

A console based Java application for managing students and their grades. Built as part of my journey getting back into Java development.

## Features

- Add students with name and grade
- View all students in the class
- Calculate class average
- Find highest and lowest grades
- Letter grade conversion (A-F)
- Full input validation with friendly error messages
- Menu driven interface

## Technologies Used

- Java 25
- IntelliJ IDEA
- Object Oriented Programming

## How to Run

1. Clone this repository
2. Open the project in IntelliJ IDEA
3. Run `Main.java`

## Project Structure

- `Student.java` — Represents a single student with name, grade and letter grade conversion
- `GradeManager.java` — Manages a list of students and provides calculations
- `Main.java` — Console interface with menu system

## What I Learned

- Object Oriented Programming principles (classes, constructors, methods)
- Encapsulation with private properties and public getters
- ArrayList for dynamic data storage
- `switch` statements for menu logic
- `do while` loops for repeated input
- Method overriding with `toString()`
- Exception handling with `try/catch`
- Defensive programming and input validation
